Media tags in HTML

Media tags are used to embed media files like images, audio, and video in HTML documents.

Image Tag

The <img> tag is used to embed images in HTML documents.

Audio Tag

The <audio> tag is used to embed audio files in HTML documents.

Video Tag

The <video> tag is used to embed video files in HTML documents.
